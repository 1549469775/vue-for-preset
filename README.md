# vue-for-preset

自己搭建的基于Vuecli 3.0脚手架的preset预设配置

### 安装 vue-cli 3.0
npm install -g @vue/cli

### 根据远程 preset 创建项目
vue create --preset 1549469775/vue-for-preset vue-project
### or
vue create --preset direct:https://github.com/1549469775/vue-for-preset.git vue-project --clone
